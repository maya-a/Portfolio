SockCiety

Link

Find a pair to your lonely sock

* this is a demo website, hosted on Heroku, Therefore the page may take up to 30 seconds to initially load.


about:
SockCiety, my first team project. This fun project has a special place in my heart, as it solves an important phenomenon we all have faced, losing socks! This website won’t help you prevent your next loss, but will provide a solution to deal with it. This website offers you to find your sock’s pair. On the website, you can view all existing socks and search for your specific sock based on its characteristics. If you find a sock you'd like you can view its location and the reviews other users gave to its owner. Then, you can buy the sock and enjoy your favorite socks again.

As mentioned above, this project has been made by a team. My main role in the project was working on the Backhand (behind the design, making the logic work). My main contribution to the design was the home page and the purchase confirmation message.

Joyage

Link

Find the perfect flight for you and your friends from around the world

* this is a demo website, hosted on Heroku, Therefore the page may take up to 30 seconds to initially load.


about:
This website has a very unique goal. Helping friends from around the world meet together on a budget. The website provides different possible destinations based on the user input and gives affordable flight tickets from all origins. We were able to do this by implementing the Amadeus API (an Application Programming Interface) in our website. This way, a group of friends from around the world can find a place to meet and book the flights in only a few minutes.

As mentioned above, this project has been made by a team. My main role in the project was working on the Backhand (behind the design, making the logic work). This includes implementing a flight source in our code and organizing the data given from this source into something presentable for the user. My main design contribution to this project was designing the logo and creating the "About" page.
